## 🚀 Quick Start

```bash
# Clone and install
git clone https://github.com/Omkar0612/AutonomOS.git
cd AutonomOS

# Backend setup
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env

# Frontend setup
cd ../frontend
npm install
cp .env.example .env

# Run (2 terminals)
# Terminal 1: cd backend && source venv/bin/activate && python3 main.py
# Terminal 2: cd frontend && npm run dev

# Open http://localhost:3000
```
